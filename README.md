# EE444-Project
Project codebase for EE444 at University of Alaska Fairbanks.

Git Cheatsheet:
```bash
git add file1, file2 # Stage files to a commit
git commit -m "Feat: Example message" # This will commit your changes to the current branch
git push # Push the branch with the new commit
git checkout branch # checkout a branch (update all files to the branch)
git fetch # Grab updated branches from the server
git pull # Pull updated code from branch
git merge branch # Merge a branch into your current branch
``` 

Example:
```bash
git add examplefile.c
git commit -m "Feat: update examplefile"
git push
```
